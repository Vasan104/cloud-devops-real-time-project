# Web Application Setup (Flask)

This document explains the creation and deployment of a basic Flask web application on AWS EC2.

## Technology Used
- Python 3
- Flask framework
- AWS EC2 (Ubuntu 22.04)

## Application Structure
- app.py: Main Flask application
- templates/index.html: UI page
- requirements.txt: Python dependencies

## Deployment Steps
- Installed Python and pip
- Created Flask application
- Opened port 5000 in EC2 security group
- Ran application using Python

## Result
The Flask application is accessible via browser and confirms successful application deployment on EC2.
